# space_cat
우주 고양이 키우기 게임

콘솔 창으로 플레이 가능한 게임으로 우주 여행을 하던 중 미지의 행성에 불시착한 우주 고양이와 플레이어가 함께 생존해가는 이야기이다.
엔지니어인 우주 고양이가 우주선을 수리하는 동안 플레이어가 고양이를 보조하여 생존하도록 돕는 것이 목적이다.
시스템 상 허기/산소/기분 세 가지 항목이 존재하며 각 항목의 수치가 모두 25 이하가 되면 게임 오버가 된다.
우주선의 수리도를 상단에 표시하여 100퍼센트를 달성하기 전까지 제한 시간 내에 생존하도록 유도한다.

주요 기능으로는 고양이 상태 관리 / 아이템 및 인벤토리 관리 / 미니게임: 리듬게임, 행성 피하기 가 있다.
기본이자 메인 기능은 고양이의 상태 수치를 회복하는 고양이 상태 관리 기능이고, 미니게임을 통해 획득한 아이템을 적재적소에 사용하여 생존해야한다.
